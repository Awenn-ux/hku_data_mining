# 知识库文档目录

将需要导入的 PDF、DOCX、TXT 文档放置在此目录下，然后使用管理脚本批量导入。

## 使用方法

### 1. 准备文档
将知识库文档（如学校政策、课程信息、FAQ 等）复制到此目录：
```
backend/knowledge_base/
├── HKU学生手册.pdf
├── 课程大纲.docx
├── 常见问题.txt
└── ...
```

### 2. 批量导入
在 `backend` 目录执行：
```bash
python admin_import_documents.py knowledge_base/
```

### 3. 验证导入
导入完成后，启动后端服务，通过智能对话测试知识库检索是否正常。

## 文档要求

- **支持格式**：PDF、DOCX、TXT
- **文件大小**：建议单个文件不超过 10MB
- **文档质量**：确保文档是可提取文本的（扫描版 PDF 需先 OCR）
- **命名规范**：使用有意义的文件名，便于后续管理

## 注意事项

- 导入的文档会自动分块并向量化，无需人工处理
- 已导入的文档信息存储在数据库中，向量数据存储在 `storage/chroma/`
- 如需更新文档内容，建议先删除旧版本再重新导入
- 大批量导入可能需要较长时间，请耐心等待

## 示例文档

建议导入的知识库文档类型：
- 学校政策与规章制度
- 课程介绍与选课指南
- 学生生活指南
- 校园设施介绍
- 学术资源使用说明
- 常见问题解答（FAQ）
